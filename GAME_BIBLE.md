## 1. IDENTITY

### 1.1 Core Concept
**Hyperreality Key** ist ein philosophisches Critical-Thinking-Spiel. Der Spieler lernt, Manipulation in Social-Media-Posts zu erkennen — mit Werkzeugen aus Philosophie, Kognitionspsychologie und Medienwissenschaft.

### 1.2 Genre
- **Primär**: Educational Game / Serious Game
- **Sekundär**: Puzzle / Pattern Recognition
- **Vibe**: Cyberpunk × Akademisch × Neon-Noir

### 1.3 Target Audience
- **Primär**: Erwachsene (18–40) mit Interesse an Medienkompetenz
- **Sekundär**: Schüler/Studenten (16+) im Bildungsbereich
- **Tertiär**: Journalisten, Pädagogen, Medienwissenschaftler

### 1.4 One-Sentence Pitch
> *"A philosophical game against disinformation — learn to spot manipulation using the tools of philosophers, psychologists, and media theorists."*

## 2. NARRATIVE & LORE

### 2.1 The World
In einer nahen Zukunft ist die Informationslandschaft kollabiert. Social Media ist ein Schlachtfeld aus Bots, Deepfakes, Echokammern und algorithmischer Manipulation. Die Wahrheit existiert noch — aber sie ist fragmentiert, versteckt hinter 12 Schichten der Täuschung.

Du bist ein **Critical Thinker** — rekrutiert von einem Rat der größten Denker der Geschichte. Deine Mission: Durchschaue die Manipulation in viralen Posts, eine Ebene nach der anderen.

## 3. LEVEL STRUCTURE

### 3.1 Progression System

```
Level 1:   3 Tools verfügbar     → 1 Post → 1 Question
Level 2:   4 Tools verfügbar     → 1 Post → 1 Question
Level 3:   5 Tools verfügbar     → 1 Post → 1 Question
Level 4:   6 Tools verfügbar     → 1 Post → 1 Question
Level 5:   7 Tools verfügbar     → 1 Post → 1 Question
Level 6:   8 Tools verfügbar     → 1 Post → 1 Question
Level 7:   9 Tools verfügbar     → 1 Post → 1 Question
Level 8:  10 Tools verfügbar     → 1 Post → 1 Question
Level 9:  11 Tools verfügbar     → 1 Post → 1 Question
Level 10: 12 Tools verfügbar     → 1 Post → 1 Question
Level 11: 12 Tools verfügbar     → 1 Post → 1 Question
Level 12: 12 Tools verfügbar     → 1 Post → 1 Question
```

**Design Principle:** Jeder Level schaltet genau EIN neues Tool frei. Der Spieler hat Zeit, jedes Tool kennenzulernen, bevor das nächste kommt.

### 3.2 Level Flow (User Journey)

```
Splash Screen
  ↓ Klick "Start"
Header: Logo + Level + Score
  ↓
Article Card (Social Media Post)
  ↓
Tool Grid (n Tools — nur aktive sind klickbar)
  ↓ Spieler wählt Tools
Highlight Overlay (markierte Wörter)
  ↓ Spieler klickt "Analyze"
A/B/C/D Question
  ↓ Spieler wählt Antwort
Feedback + Explanation (Richtig/Falsch)
  ↓ Klick "Next Level →"
Nächstes Level (remount)
  ...
Level 12 → Victory Screen
  ↓
Gesamtscore + Statistiken + Restart
```

### 3.3 Scoring
- **Correct answer**: +10 Punkte
- **Wrong answer**: 0 Punkte
- **Max Score**: 120 Punkte (12 × 10)
- **Rank system** (in future):
  - 0-30: Novice
  - 40-70: Critical Thinker
  - 80-100: Truth Seeker
  - 110-120: Hyperreality Master

### 3.4 Difficulty Curve
- **Levels 1-4**: Einfach — klare Manipulation, offensichtliche Tools
- **Levels 5-8**: Mittel — subtilere Manipulation, Tools müssen kombiniert werden
- **Levels 9-12**: Schwer — mehrdeutige Posts, alle Tools verfügbar

## 5. DESIGN PHILOSOPHY

### 5.1 Golden Rules
1. **Show, don't tell** — Jedes Konzept wird an einem echten/realistischen Post demonstriert
2. **Real articles only** — Keine erfundenen Beispiele. Echte Quellen (Snopes, NPR, Guardian, Forbes, BBC, AP)
3. **One skill per level** — Fokus. Nicht überladen.
4. **Immediate feedback** — Nach jeder Antwort: Erklärung, warum richtig/falsch
5. **Juicy but purposeful** — Animationen und Neon-Stil dienen der Immersion, nicht der Ablenkung

### 5.2 What Makes Hyperreality Key Different
- **Philosophisch fundiert** — Nicht "rate was ich denke", sondern echte Theorien
- **Tool-basiert** — Der Spieler sammelt Werkzeuge, nicht Punkte
- **Realwelt-Relevanz** — Jeder Level trainiert eine Fähigkeit, die im echten Leben nützlich ist
- **Kein Grinding** — 12 Levels, fokussiert, keine Filler-Inhalte

## 6. LORE BIBLE (Expanded Narrative)

> *Eingefügt von Konstantin — kann erweitert werden*

### 6.1 The Fall
Im Jahr 2024 erreichte die Informationskrise einen Wendepunkt. KI-generierte Inhalte machten 60% des Social-Media-Traffics aus. Echtheitszertifikate versagten. Die Menschen zogen sich in ihre Echokammern zurück. Die Wahrheit wurde zum Luxusgut.

### 6.2 The Awakening
Eine kleine Gruppe von Denkern — die letzte Generation, die noch eine Welt vor der Hyperrealität kannte — begann, ein Arsenal an Werkzeugen zu entwickeln. Nicht zur Bekämpfung, sondern zur **Durchschauung** der Täuschung.

### 6.3 The Key
Die Werkzeuge wurden in einem einzigen Konzept vereint: dem **Hyperreality Key**. Ein Framework aus 12 analytischen Linsen, die zusammen die Fähigkeit verleihen, jede Form von Manipulation zu erkennen.

### 6.4 The Player
Du bist der erste Träger des Schlüssels. Nicht ein Held — sondern ein **critical thinker**. Jeder gemeisterte Level ist ein Zahn mehr im Schlüssel.

## 8. GLOSSARY

| Term | Definition |
|------|-----------|
| **Hyperreality** | Zustand, in dem Simulation und Realität nicht mehr unterscheidbar sind (Baudrillard) |
| **Simulacrum** | Eine Kopie ohne Original (Baudrillard) |
| **Myth** | Naturalisierte Geschichte — kulturelle Konstruktion als Natur dargestellt (Barthes) |
| **Echokammer** | Geschlossener Kommunikationsraum, in dem nur bestätigende Informationen zirkulieren |
| **Moral Foundations** | 6 angeborene moralische Intuitionen (Haidt) |
| **Agenda Setting** | Die Macht der Medien, nicht WAS wir denken, sondern WORÜBER wir nachdenken |
| **Red Herring** | Ablenkungsmanöver — vom eigentlichen Thema wegführen |
| **False Appeal** | Fälschliche Berufung auf Autorität, Tradition, oder Beliebtheit |
