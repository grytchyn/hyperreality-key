## 1. DIE WELT: "Hyperreality 2037"

### Was ist passiert?

Berlin, 2037. Seit 2032 tragen Menschen *Lenses* — AR-Kontaktlinsen, die die physische Realität mit Daten überlagern. Jeder Quadratmeter Sicht ist ein Interface: Schlagzeilen schweben über Häuserfassaden, Produktbewertungen überlagern Geschäfte, Nachrichten-Overlays füllen die Luft wie zweite Haut.

Man nennt es **die Hyperrealität**: Die Karte (Daten-Overlay) hat das Territorium (physische Realität) ersetzt. Nicht dystopisch — *pragmatisch*. Warum die echte Welt sehen, wenn die augmentierte Version nützlicher, schöner und besser kuratiert ist?

Die Lenses sind kostenlos. Bereitgestellt von *NexusSphere* — einem globalen AR-Netzwerk-Betreiber, der aus einer Non-Profit-Idee (2019) zum mächtigsten Infrastruktur-Konzern wurde.

Doch unter der glatten Oberfläche gibt es **Das Rauschen** — Lücken im Overlay, flackernde Widersprüche, Momente in denen zwei Nachrichten-Streams kollidieren.

### Die Spieler-Rolle: Reality-Archäologe

Du bist kein Held. Du bist ein **Reality-Archäologe** — ein freiberuflicher Medienanalyst mit einer modifizierten Lens-Firmware. Dein philosophischer Werkzeugkasten lässt dich die Schichten der Hyperrealität durchdringen.

> *Die meisten Menschen fragen: „Ist das wahr?"*
> *Ein Reality-Archäologe fragt: „Wer hat diese Wahrheit gebaut — und warum?"*

### Die 4 Ebenen der Spielwelt

| Ebene | Name | Visuell | Spieler-Einsicht |
|-------|------|---------|------------------|
| **1** | Nachrichten-Oberfläche | Sauber, hell, Apple-Bloomberg. Frosted Glass, sanfte 3D-Perspektive | "So sieht die Welt für alle aus" |
| **2** | Das Rauschen | Glitch-Art + Netzwerk-Topologie. Neon-Lila, Rot, Gelb, Cyan | "Hier liegen die Strukturen verborgen" |
| **3** | Serverraum der Algorithmen | Westworld-Kontrollräume trifft Tron. Monolithische Licht-Blöcke | "So arbeiten die Algorithmen" |
| **4** | Hyperreales Zentrum | Endlose Spiegelhalle. Weich, neblig, selbstreferenziell | "Hier simulieren Simulationen sich selbst" |

## 2. PROTAGONIST & NARRATIVE

### Eröffnungsgeschichte (Intro, 30 Sekunden)

> *Berlin, 14. Juni 2037. 07:42 Uhr.*
>
> Du setzt deine Lenses auf. Die Welt erwacht in Schichten: Wetterdaten oben links, deine Agenda rechts, eine Push-Nachricht von NexusFeed blinkt: **"Sicherheitslücke in Berliner Trinkwasser entdeckt"**.
>
> Du öffnest den Artikel. Alles sieht perfekt aus — Zitate von Experten, offizielles Siegel. Aber etwas flackert. Ein Riss im Overlay. Hinter der Nachricht siehst du für einen Moment die rohe Realität.
>
> Du schaltest dein Analyse-Interface ein. Und plötzlich siehst du nicht mehr die Nachricht. Du siehst ihre Architektur: Die Machtlinien, die sie formen. Das Spektakel, das sie inszeniert. Die narrative Maschinerie.
>
> **Willkommen. Du siehst die Welt, wie sie wirklich funktioniert.**

### Narrative Struktur durch die 7 Tiers

| Tier | Name | Welt-Ebene | Spieler-Einsicht | Neues Feature |
|------|------|------------|------------------|---------------|
| **1** | 👁️ Die Oberfläche | Ebene 1 | "Manche Nachrichten sind manipuliert" | 4 Basis-Werkzeuge |
| **2** | 🔍 Der Blick | Ebene 1→2 | "Ich kann Techniken erkennen" | Highlighting + Sorting |
| **3** | 🧠 Das Muster | Ebene 2 | "Ich sehe wiederkehrende Muster" | Timeline-Construction |
| **4** | 🏛️ Die Struktur | Ebene 3 | "Die Muster sind in Algorithmen eingebaut" | Serverraum zugänglich |
| **5** | 🌐 Das Netz | Ebene 3→4 | "Alles hängt mit allem zusammen" | Multi-Quellen-Fälle |
| **6** | 🌀 Die Simulation | Ebene 4 | "Manche simulieren nur sich selbst" | Meta-Analyse |
| **7** | 🔑 Der Schlüssel | Alle | "Ich kann die Hyperrealität lesen" | Vollständiger Werkzeugkasten |

### Die 3 Narrative Hooks für die ersten Minuten

1. **Der widersprüchliche Push** — Zwei Push-Nachrichten zum selben Ereignis sagen das Gegenteil. Welche ist echt?
2. **Das flackernde Denkmal** — Ein Denkmal hat plötzlich ein neues AR-Overlay mit einer anderen Geschichte.
3. **Der Algorithmus, der dich kennt** — Eine Nachricht, die exakt auf deine Ängste zugeschnitten ist. Unheimlich präzise.

## 3. CORE GAME-LOOP

### Der Grund-Zyklus (3-7 Minuten)

```
1. Nachricht empfangen (Push, Feed, Fallakte)
       ↓
2. Überfliegen — Welche Art von Manipulation?
       ↓
3. Philosophisches Werkzeug auswählen
       ↓
4. Highlighten / Kategorisieren / Verbinden
       ↓
5. Analyse bestätigen → Feedback + Erkenntnis
       ↓
6. Score + neues Werkzeug / Tier-Fortschritt
```

### 5 Core Loops im Detail

**Loop A — Der Scanner (Tutorial/Tier 1)**
- Nachricht scannen → manipulative Passagen highlighten → Technik benennen
- Dauer: 2-3 Min.
- Ideal für erste 4 Werkzeuge

**Loop B — Der Detektiv (Kern-Gameplay, Tier 2-4)**
- Fallakte erhalten → Quellen sammeln → Highlighten + Kategorisieren → Timeline rekonstruieren → Bericht verfassen
- Dauer: 10-15 Min.
- Kombiniert 3-5 Werkzeuge

**Loop C — Der Strateg (Fortgeschritten, Tier 3-5)**
- Werkzeug-Deck zusammenstellen → Nachrichtenwelt erkunden → Entscheidungen treffen (welcher Quelle folgen?) → Konsequenzen erleben
- Dauer: 15-20 Min.
- Roguelike-Struktur: Jeder Run anders

**Loop D — Der Chronist (Spezial, Tier 4-6)**
- Mehrere Quellen zu einem Ereignis → Widersprüche erkennen → wahre Timeline rekonstruieren
- Dauer: 8-12 Min.
- Mustererkennung über mehrere Quellen

**Loop E — Der Skeptiker (Boss, Tier 5-7)**
- Komplexer Fall mit allen Werkzeugen → Desinformationskampagne vollständig analysieren → abschließendes Urteil
- Dauer: 20-25 Min.
- Abschluss jedes Skeleton (Werkzeug-Gruppe)

### Session-Struktur (30-45 Min.)

```
WARM-UP (3-5 Min.) — Loop A
    ↓
HAUPTFALL (15-20 Min.) — Loop B, C, oder D
    ↓
BOSS-PHASE (5-8 Min.) — Komplexe Entscheidung
    ↓
COOLDOWN (3-5 Min.) — Ergebnisse, Belohnungen, neues Werkzeug
```

## 4. PHILOSOPHISCHE WERKZEUGE (9 Skeletons)

### Skeleton 1: Hermeneut 📖
**Kern:** Talmudische Middot + Gadamer
**Tool:** Pluralismus-Brille — Ein Text kann mehrere Bedeutungen haben
**Freigeschaltet:** Tier 1

### Skeleton 2: Skeptiker 🔍
**Kern:** Pyrrhon + Descartes + Popper
**Tool:** Zweifel-Hebel — "Woher wissen wir das?"
**Freigeschaltet:** Tier 1

### Skeleton 3: Rhetor 🎭
**Kern:** Aristoteles
**Tool:** Persuasion-Detektor — Ethos/Pathos/Logos + Trugschlüsse
**Freigeschaltet:** Tier 1

### Skeleton 4: Semiotiker 🗺️
**Kern:** Saussure + Peirce + Barthes
**Tool:** Zeichen-Lupe — Signifikant/Signifikat, Denotation/Konnotation
**Freigeschaltet:** Tier 2

### Skeleton 5: Archäologe 🏛️
**Kern:** Foucault + Nietzsche
**Tool:** Macht-Hammer — Diskurs, Macht/Wissen, genealogische Tiefe
**Freigeschaltet:** Tier 2

### Skeleton 6: Ökonom 💰
**Kern:** Marx + Chomsky + Bourdieu
**Tool:** Kapital-Scanner — Wem gehört das Medium?
**Freigeschaltet:** Tier 3

### Skeleton 7: Dekonstrukteur ✂️
**Kern:** Derrida + Lyotard
**Tool:** Binär-Skalpell — Différance, binäre Oppositionen, Meta-Erzählungen
**Freigeschaltet:** Tier 4

### Skeleton 8: Rhizomatiker 🔗
**Kern:** Deleuze + Guattari + Latour
**Tool:** Netzwerk-Kompass — Falsche Verbindungen trennen, echte knüpfen
**Freigeschaltet:** Tier 5

### Skeleton 9: Hyperrealist 🌀
**Kern:** Baudrillard + Debord + Han
**Tool:** Simulations-Index — 4 Ordnungen, Spektakel-Erkennung
**Freigeschaltet:** Tier 6

### Meta-Schlüssel (Tier 7): Hyperreality Key 🔑
**Alle 9 Skeletons integriert**
→ Volle Analyse-Fähigkeit für jede Nachricht

## 5. LEVEL-DESIGN & PROGRESSION

### Level-Architektur

| Tier | Levels | Werkzeuge | Mechanik | Dauer |
|------|--------|-----------|----------|-------|
| 1 | 3× Tutorial | 4 Basis (Sokrates, Aristoteles, Kahneman, Elenchus) | Highlighting | 15 Min. |
| 2 | 4× Anfänger | +2 (Barthes, Foucault) | + Sorting | 40 Min. |
| 3 | 3× Aufbau | +3 (Marx, Derrida, Lyotard) | + Timeline | 45 Min. |
| 4 | 3× Mittel | +2 (Deleuze, Latour) | + Multi-Quelle | 45 Min. |
| 5 | 3× Fortgeschritten | +3 (Baudrillard, Debord, Han) | + Strategie-Runs | 60 Min. |
| 6 | 2× Meister | +2 (Zuboff, Nietzsche) | + Meta-Analyse | 50 Min. |
| 7 | 2× Final | 1 (Hyperreality Key) | + Alle kombiniert | 60 Min. |

**Gesamt: 20+ Levels / ~6-8 Stunden Kerngameplay**

### Dynamische Schwierigkeitsanpassung (DDA)

- **Tier 1-2:** Nur offensichtliche Manipulationen; 80%+ Erfolgsrate
- **Tier 3-4:** Mehrdeutige Fälle; subtile kognitive Verzerrungen; 60-70% Erfolgsrate
- **Tier 5-6:** Kombinationen mehrerer Verzerrungen; 40-50% Erfolgsrate
- **Tier 7:** Echtzeit-Analyse aktueller Nachrichten ohne vorgegebene Optionen

**Regel:** 3 richtige Analysen in Folge → automatischer Aufstieg. 2 falsche → Wiederholung mit einfacherem Material.

## 6. RETENTION & ENGAGEMENT (Ethisch)

### Erlaubte Systeme ✅

| System | Wie | Warum ethisch |
|--------|-----|---------------|
| **Streak-System** | Tägliche Login-Boni (eine Nachricht des Tages + EP) | 1 Joker-Tag/Woche; alle Inhalte nachspielbar |
| **Tägliche Herausforderung** | Nachricht des Tages (3 Min.) + Bonus-EP | Keine Skip-Mikrotransaktionen |
| **Abzeichen** | "Lügenentlarver" (10 Techniken korrekt identifiziert) | Dokumentieren Fähigkeiten, nicht Konsum |
| **Season-Pass Light** | Kosmetische Themes (Buero-Hintergrund, Avatar) | Nur kosmetisch; dauerhaft erwerbbar |
| **Skill-Tree** | Sichtbarer Fortschritt durch die 18 Werkzeuge | Vermittelt Kompetenz-Gefühl |

### VERBOTENE Systeme ❌

| Pattern | Warum verboten | Alternative |
|---------|----------------|-------------|
| **Energie-System** | Führt zu Frustration oder Paywall | Natürliche Sessions-Grenzen |
| **Lootboxen** | Glücksspiel-Mechanik, manipulativ | Feste Belohnungen pro Level |
| **Pay-to-Win** | Untergräbt Bildungsauftrag | Einmaliger Kauf oder Freemium |
| **FOMO-Timer** | Künstliche Dringlichkeit, Angst | "Tool des Tages" ohne Verfallsdatum |
| **Endloses Scrollen** | Sucht-Optimierung, +40-60% Session-Zeit | Klare Sessions-Grenzen |

### Warum das funktioniert (SDT)

| Bedürfnis | Umsetzung |
|-----------|-----------|
| **Autonomie** | Spieler wählt Pfad durch 18 Werkzeuge; keine lineare Zwangsführung |
| **Kompetenz** | Progressive Tiers mit steigender Komplexität; sofortiges Feedback |
| **Verbundenheit** | Analyse-Ergebnisse teilbar; Diskussion von Fällen |

## 7. UI/UX — Mobile-First

### Spezifikationen

| Anforderung | Umsetzung |
|-------------|-----------|
| Buttons | ≥ 48×48 DP (Google Material Design) |
| Navigation | Swipe-Interaktionen (links/rechts für Analyse-Entscheidungen) |
| Daumen-Zonen | Interaktive Elemente im unteren Drittel |
| Start-Design | 360×780 px (Smartphone), Skaliert hoch |
| PWA | Installierbar ohne App Store |
| Max Optionen | 2-3 pro Entscheidungspunkt |

### Bildschirm-Aufbau (immer sichtbar)

```
┌─────────────────────────────┐
│  🔑 MISSION: "Versteckte   │
│     Agenda"     ★  Score    │
├─────────────────────────────┤
│                             │
│   🗞️ ARTIKEL               │
│   [markierte Textstellen]   │
│   [Hover-Tooltips]          │
│                             │
├─────────────────────────────┤
│ 🛠️  🌀  🎭  👁️  📖       │
│   ████████░░░░ 3/4 Hinweise│
└─────────────────────────────┘
```

### Flow für den Spieler

1. **Start:** Intro-Text + "Neue Nachricht empfangen" Button
2. **Lesen:** Nachricht erscheint als AR-Overlay (News-Design)
3. **Analysieren:** Wähle philosophisches Werkzeug → Text wird markiert → bestätige
4. **Erkennen:** Nach jeder Analyse: "Dein kognitiver Blind Spot heute war X"
5. **Abschließen:** Nach 4 richtigen Analysen: "Was ist die versteckte Agenda?"
