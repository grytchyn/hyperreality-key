# Hyperreality Key — Tool Distillation

> Game Designer's Note: 18 Tools = Interface Overload. Auf 7 CORE-Tools geschnitten + Info-Plates + Deep-Dives.

## Die 7 CORE-Tools (Main Toolbar)

```
TUTORIAL → ⚡ Fallacy Scanner
              ↓
         🎭 Aristotle Rhetoric
              ↓
         🧠 Cognitive Bias
              ↓
         🗺️ Barthes Meaning Map
              ↓
         ⚔️ Derrida Binary Scalpel
              ↓
         📊 Haidt Moral Foundations
              ↓
         🌀 Baudrillard Simulacrum Meter (CLIMAX)
              ↓
         🔬 Deep Analysis Menu (5 Deep-Dives)
```

### Tool 1: ⚡ Fallacy Scanner
*"Findet die Denkfehler, bevor sie dich fangen."*
- **UI-Effekt:** Rote Highlights + inline Badge + Severity-Dot
- **Warum #1:** Jeder versteht "logischer Fehlschluss" sofort

### Tool 2: 🎭 Aristotle Rhetoric
*"Wird hier dein Kopf oder dein Herz angesprochen?"*
- **UI-Effekt:** 3-Farb-Heatmap 🔵Ethos/🔴Pathos/🟩Logos + Layer-Toggles
- **Warum:** **Spektakulärste Visualisierung** — Artikel auf einen Blick erkennbar

### Tool 3: 🧠 Cognitive Bias
*"Dein Gehirn liebt Abkürzungen. Hier sind sie."*
- **UI-Effekt:** Gelb-Amber Highlights + Hover-Tooltips + Density-Bar
- **Warum:** Persönlich — zeigt, wie *dich* der Text manipuliert

### Tool 4: 🗺️ Barthes Meaning Map
*"Was steht da wirklich? Und was wird dir verkauft?"*
- **UI-Effekt:** 3-stackable Overlays (Denotation → Konnotation → Mythos)
- **Warum:** **Kinematografisch** — Layer-Peel mit "Aha!"-Garantie

### Tool 5: ⚔️ Derrida Binary Scalpel
*"Jede Geschichte baut auf Gegensätzen. Finde sie."*
- **UI-Effekt:** 🔴🔵 Kontrast-Paare + Verbindungslinien + Deconstruction Mode
- **Warum:** Einzigartiger Effekt — zeigt US vs THEM Architektur

### Tool 6: 📊 Haidt Moral Foundations
*"Welche moralischen Knöpfe werden hier gedrückt?"*
- **UI-Effekt:** 6 Pastellfarben + Radar-Chart
- **Warum:** Politisch aufschlussreich — Linke/Rechte aktivieren verschiedene Moral-Basics

### Tool 7: 🌀 Baudrillard Simulacrum Meter
*"Wie real ist das, was du da liest?"*
- **UI-Effekt:** 4-Segment-Gauge 🟢🟡🟠🔴 + Text-Fade bei Order 4 (Dissolve-Animation)
- **Warum:** **Das Identity-Tool** — Name des Spiels. Klimaktische Enthüllung
